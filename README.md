# Vega

Vega is a open source web based video editor. 

This is an experimental project.

Editor - https://vega.toshusai.net

Documents - https://vega.toshusai.net/docs

<img src="static/static/img/vega_screenshot.png" width="100%">

# Support Browsers
- Google Chrome 92
